This project explores a method for transmitting data through cellular networks by embedding it within continuous speech. Using the NTIMIT dataset as a standard continuous speech corpus, we map hexadecimal data representations to specific voice segments stored in a lookup table. The audio is then passed through GSM codecs (FR, EFR, AMR) to simulate real network conditions. The goal is to recover the original message with minimal distortion using FFT-based decoding techniques. Results show near-perfect accuracy in simulated environments (100\% direct, 99\% with GSM codecs) and 20-54\% accuracy in practical SIP calling implementations
